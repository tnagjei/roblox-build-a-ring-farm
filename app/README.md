# app
- 用途：Next.js App Router 页面、全局样式和静态 SEO 辅助路由。
- 关键入口：`page.tsx`、`layout.tsx`、`sitemap.ts`、`robots.ts`
- 边界/依赖：页面只读 `lib` 和 `components`，旧图片路径由根 `next.config.mjs` 跳转到 `public` 静态资源。
> 一旦本目录内容变化，请更新本文件
